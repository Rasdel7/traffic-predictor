# Traffic Congestion Predictor 🚦

Smart city traffic analytics app predicting
congestion levels across Indian cities.

## Live Demo
[Click here](YOUR_STREAMLIT_URL)

## Features
- Random Forest congestion classifier
- 4 levels: Free Flow, Light, Moderate, Heavy
- 14 input features including weather, accidents
- Hourly traffic heatmap (Day × Hour)
- City traffic map with hotspot markers
- Best time to travel advisor
- Route A vs B comparison
- Time saved estimation
- Weather impact analysis
- Feature importance visualization

## Cities Covered
Bhubaneswar, Mumbai, Delhi, Bangalore,
Chennai, Hyderabad, Kolkata, Pune

## Tools Used
- Python, Streamlit, Scikit-learn,
  Plotly, Pandas, NumPy

## How to Run Locally
pip install streamlit pandas numpy plotly scikit-learn
streamlit run app.py
